# 🚀 Vocl Quick Start Guide

## 📱 How to Run the App in 3 Simple Steps

### Step 1: Navigate to the app directory

```bash
cd /c/Users/vashl/source/repos/vocl/vocablens
```

### Step 2: Start the development server

```bash
npx expo start --port 3000
```

*If port 3000 is busy, try another port like 4000, 5000, etc.*

### Step 3: Scan the QR code

1. **Install Expo Go** on your phone:
   - [Android - Google Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)
   - [iOS - App Store](https://apps.apple.com/us/app/expo-go/id982107779)

2. **Open Expo Go** and scan the QR code displayed in your terminal

3. **Make sure your phone and computer are on the same WiFi network**

## 🎯 What to Test

### 1. Create a Workspace
- Tap "Create Workspace"
- Enter a name (e.g., "Spanish Learning")
- Select source language (e.g., "English")
- Select target language (e.g., "Spanish")
- Tap "Create"

### 2. Add Vocabulary
- Navigate to your workspace
- Tap "Add Vocabulary"
- Enter word and translation (e.g., "Hello" → "Hola")
- Add notes if desired
- Tap "Save"
- Add at least 10 items for a full challenge

### 3. Take the Daily Challenge
- Tap "Daily Challenge"
- Answer the 10 flashcard questions
- See immediate feedback on each answer
- Complete all 10 to save your session
- Watch your streak increase!

### 4. Track Your Progress
- View your current streak
- See your longest streak
- Check vocabulary weights (higher = needs more practice)
- Monitor your accuracy over time

## 🔧 Troubleshooting

### Port already in use?
```bash
npx expo start --port 4000
```

### App not starting?
```bash
npx expo start --clear
```

### Need to verify files?
```bash
node simple-test.js
```

### Want to generate QR code?
```bash
node generate-qr.js
```
